# 爆品工作台 Edge 插件

这个插件用于读取当前浏览器页面中已经渲染出来的商品卡片，并发送到本地爆品广告工作台。

## 安装

1. 打开 Edge，进入 `edge://extensions/`。
2. 开启“开发人员模式”。
3. 点击“加载解压缩的扩展”。
4. 选择本目录：`E:\5.18\new-chat\edge-extension`。

## 使用

1. 先启动工作台：`node src/server.js`。
2. 打开 `http://localhost:4173`。
3. 在 Edge 打开你有权访问的商品列表页，等待商品加载出来。
4. 点击浏览器工具栏里的“爆品工作台采集助手”。
5. 点击“发送到工作台”。
6. 回到工作台，点击左侧“读取插件数据”。

插件只读取当前页面已渲染的 DOM 内容，不处理登录态，不绕过平台限制，不做后台爬取。请只在你有权访问、复制或使用的数据源上使用，并保留授权记录和页面截图。
