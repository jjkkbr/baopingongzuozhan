# v0.1.0 测试版发布草稿

## 发布定位

这是“爆品广告工作台”的首个 Windows 测试版，用于内部试用和流程验证。它是合规优先的本地桌面工具，不是爬虫项目，也不提供未授权页面抓取或竞品素材搬运能力。

## 下载文件

建议在 GitHub Release 附件中上传：

```text
release/爆品广告工作台 Setup 0.1.0.exe
```

本地构建信息：

- 版本：`0.1.0`
- 安装包大小：约 76 MB
- 构建时间：2026-06-10
- 构建命令：`npm.cmd run dist`
- 当前安装包未配置正式代码签名证书，Windows 安装时可能提示“未知发布者”。

## 主要功能

- 本地网页工作台和 Electron 桌面版。
- Edge 插件采集助手，读取当前已渲染、用户有权访问和使用的商品页面 DOM 内容。
- 授权 CSV、授权文本/HTML、历史销量 CSV/API 导入。
- 商品预览、销量榜、指标卡和真实历史销量趋势展示。
- 素材结构分析、AI 素材洞察、视觉风险工作包和人工复核清单。
- 原创短视频脚本、海报方案、分镜和广告方案生成。
- 创意结果质量控制，包含质量分、发布闸门、维度评分和改写简报。
- 合规审核、敏感词/品牌规则库、品牌授权状态和品牌授权范围。
- 授权凭证台账和素材版权台账。
- 批量生成、批量审核、历史方案复审、取消、重试、归档、任务搜索和批量报告包导出。
- 专业交付包导出，包含 `plan.json`、`plan.md`、`review-checklist.csv` 和 `manifest.json`。

## 安装和启动

1. 下载并运行 `爆品广告工作台 Setup 0.1.0.exe`。
2. 如 Windows 提示未知发布者，确认来源为本仓库 Release 后继续安装。
3. 安装完成后从桌面快捷方式或开始菜单启动“爆品广告工作台”。
4. 首次启动后按新手向导配置 AI、加载 Edge 插件并导入授权商品数据。

也可以从源码运行：

```powershell
git clone https://github.com/jjkkbr/baopingongzuozhan.git
cd baopingongzuozhan
npm install
npm.cmd start
```

桌面开发运行：

```powershell
npm.cmd run desktop
```

## AI Key 配置

- 普通阿里百炼 Key 推荐使用 `https://dashscope.aliyuncs.com/compatible-mode/v1` 和 `qwen-plus`。
- Coding Plan / 套餐专属 `sk-sp` Key 使用 `https://coding.dashscope.aliyuncs.com/v1` 和 `qwen3-coder-plus`。
- 视觉分析建议单独配置普通百炼 Key，模型可选 `qwen-vl-plus`、`qwen-vl-max`、`qwen3-vl-plus` 或 `qwen3-vl-flash`。
- 完整 Key 只保存在本机私有配置或 Electron 用户数据目录，页面只显示尾号。
- 不要把 API Key、Cookie、Token 或登录态写入代码、README、前端文件或可提交配置。

## Edge 插件加载

1. 打开 Edge 的 `edge://extensions/`。
2. 开启“开发人员模式”。
3. 点击“加载解压缩的扩展”。
4. 选择安装目录或源码目录中的 `edge-extension/`。
5. 在有权访问和使用的商品页面点击扩展“发送到工作台”，再回到工作台点击“读取插件数据”。

插件只读取当前页面已经渲染的 DOM 内容，不处理登录态，不绕过平台限制，不做后台爬取，不自动翻页，不下载未授权素材。

## 已知问题

- 当前测试安装包未签名，Windows 可能提示“未知发布者”。
- AI 能力依赖用户自行配置可用模型和 Key；Key 类型、Base URL 和模型不匹配时会调用失败。
- Edge 插件识别率依赖真实平台页面 DOM，平台改版后需要补充授权样本并回归验证。
- 视觉风险分析是素材风险预检和人工复核辅助，不能替代法律、平台审核或版权审核结论。
- 历史销量趋势只展示真实导入字段，没有真实历史字段时不会生成模拟趋势。

## 合规边界

- 不实现绕过登录、验证码、风控、频控或反爬的能力。
- 不抓取未授权页面作为默认能力。
- 不下载、搬运、复刻竞品图片、视频、音乐、字幕、演员表演或剪辑。
- 不生成规避平台审核、规避版权或规避广告法的方案。
- 仅用于官方开放平台、商家后台导出、达人/商家授权数据、内部投放数据、用户有权访问和使用的页面内容，以及用户手动粘贴的授权文本、HTML 或 CSV。
