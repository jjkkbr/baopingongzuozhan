FIRST RUN

首次启动向导

1

配置 AI Key

选择“阿里百炼 Coding 专属”或通用接口，保存 Key 后点击“测试连接”。完整 Key 只保存在本机。

2

安装 Edge 插件

打开插件目录，在 Edge 的扩展页面加载 `edge-extension`，只读取当前已授权页面的已渲染内容。

3

导入授权商品

粘贴 CSV、授权文本/HTML，或从插件读取商品卡片。发布前保留授权记录和后台截图。

4

生成并审核

选择商品后运行分析、生成广告方案和合规审核，再导出 JSON 到本地目录。
